# React Native List Examples

This App shows examples of lists: 
- FlatList
- FlatList with StickyHeader
- Section List
- List with Images
- Grid View
- Image Gallery
- Search Example
- Infinite Pagination


## Install and run
```
git clone https://github.com/pjbelo/react-native-list-example.git
cd react-native-list-example
npm install
npm start
```
